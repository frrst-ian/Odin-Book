# Odin Book

Odin Book is a simple and minimal social media website (inspired by IG). This is my final project in the [The Odin Project](https://www.theodinproject.com/) NodeJS course.

## Demo

[Odin Book](https://odinbookxd.netlify.app/login)

## Run Locally

Clone the project and install dependencies:
```bash
npm install
```

Create a `.env` file in the root:
```
VITE_API_URL=http://localhost:3000
```

Then start the dev server:
```bash
npm run dev
```

> Requires the backend to be running. See [Odin Book Backend](https://github.com/frrst-ian/Project-Odin-Book).
